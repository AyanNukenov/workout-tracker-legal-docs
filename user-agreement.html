Пользовательское соглашение мобильного приложения "Трекер Тренировок"

1. Введение

Настоящее Пользовательское соглашение (далее - Соглашение) регулирует использование мобильного приложения "Трекер Тренировок" (далее - Приложение).

2. Термины и определения

  • Пользователь: физическое лицо, использующее Приложение
  • Контент: информация, загружаемая и создаваемая Пользователем
  • Сервис: функционал Приложения

3. Права и обязанности Пользователя

3.1. Пользователь обязуется:

  • Предоставлять достоверную информацию
  • Соблюдать законодательство, в том числе в области информационных технологий и защиты персональных данных
  • Не использовать Приложение для противоправных целей
  • Бережно относиться к своим учетным данным

3.2. Пользователь имеет право:

  • Использовать функционал Приложения
  • Создавать и редактировать личный контент
  • Удалять свой аккаунт в любой момент

4. Права и обязанности Разработчика

4.1. Разработчик обязуется:

  • Обеспечивать работоспособность Приложения
  • Защищать персональные данные Пользователя
  • Улучшать качество сервиса

4.2. Разработчик имеет право:

  • Модифицировать функционал Приложения
  • Временно приостанавливать работу сервиса
  • Блокировать аккаунт при нарушении Соглашения
  • Удалить безвозвратно аккаунт при нарушении Соглашения

5. Ответственность сторон

  • За неисполнение условий Соглашения стороны несут ответственность согласно законодательству
  • Разработчик не несет ответственности за косвенные убытки Пользователя

6. Порядок использования Приложения

  • Пользователь самостоятельно определяет интенсивность использования сервиса
  • Контент, создаваемый Пользователем, является его собственностью

7. Конфиденциальность и защита данных

  • Персональные данные защищены согласно Политике конфиденциальности
  • Пользователь согласен на обработку своих данных при регистрации

8. Заключительные положения

  • Соглашение может быть изменено в одностороннем порядке
  • Продолжение использования Приложения означает согласие с Политикой конфиденциальности и Пользовательским соглашением
