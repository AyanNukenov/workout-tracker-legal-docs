ТЕМА: Политика конфиденциальности и защиты персональных данных

I. Цель документа

А. Настоящая Политика конфиденциальности определяет:

  • Порядок сбора персональных данных
  • Цели обработки информации
  • Меры защиты персональных данных пользователей

Б. Основные принципы:

  • Законность обработки персональных данных
  • Уважение прав пользователей
  • Обеспечение безопасности личной информации

II. Собираемые персональные данные

А. При регистрации в приложении:

  • Логин пользователя
  • Адрес электронной почты
  • Зашифрованный пароль

Б. Данные, которые НЕ собираются:

  • IP-адрес
  • Информация об устройстве
  • Журналы использования приложения

В. Обработка ваших персональных данных осуществляется на основании:

  • Вашего добровольного согласия, предоставляемого при регистрации
  • Необходимости исполнения договора (Пользовательского соглашения)
  • Закона Республики Казахстан "О персональных данных и их защите"

III. Цели обработки персональных данных

А. Основные цели:

  • Предоставление функционала приложения
  • Персонализация пользовательского опыта
  • Коммуникация с пользователем
  • Улучшение качества сервиса

IV. Защита персональных данных

А. Технические меры защиты:

  • Шифрование паролей с использованием алгоритмов
  • Использование SSL/TLS для передачи данных
  • Многофакторная защита базы данных
  • Шифрование данных на уровне базы данных

Б. Ограничения доступа:

  • Строго ограниченный доступ к персональным данным
  • Защита базы данных в несколько этапов

В. Мы храним ваши персональные данные:

  • В течение всего срока действия аккаунта
  • В случае удаления Вами аккаунта персональные данные удаляются

V. Права пользователя

А. Пользователь имеет право:

  • Получать информацию о своих персональных данных
  • Удалить аккаунт
  • Отзывать согласие на обработку данных путем удаления аккаунта

Б. Удаление аккаунта:

  • Полное и безвозвратное удаление всех персональных данных
  • Невозможность восстановления после подтверждения

VI. Политика использования cookies

А. Минимальное использование cookies:

  • Только для аутентификации
  • Сохранение пользовательских настроек

VII. Передача данных третьим лицам

А. Принципы:

  • Запрет продажи персональных данных
  • Отсутствие передачи данных без согласия пользователя
  • Соблюдение законодательства Республики Казахстан, а также международных нормативно-правовых актов, ратифицированных Республикой Казахстан

VIII. Изменение политики конфиденциальности

А. Порядок внесения изменений:

  • Актуальная версия всегда доступна в Приложении
  • Возможность уведомления пользователей о существенных изменениях
